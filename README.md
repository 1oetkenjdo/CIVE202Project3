# CIVE 202 Project 3: NHTS and NGSIM Analysis

## Project Summary
This project analyzes two U.S. transportation datasets:
- **NHTS (National Household Travel Survey):** household-level transportation usage patterns.
- **NGSIM (Next Generation Simulation):** vehicle trajectory and car-following driving behavior.

The analysis includes required visualizations and an Intelligent Driver Model (IDM) simulation study to compare observed vs simulated follower behavior.
